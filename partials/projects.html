Mobile application security and Ads

For academic reasons I developed two mobile applications. One using native Java code and the other using PhoneGap. The task of the project was to expose some major security concerns regarding android applications, more specifically on how it's possible to get information about the user through mobile ads. By the end of the development process the results were very interesting. By injecting some javascript code to the ad loaded on the phone, I was able to retrieve the users location. The applications include some core mobile applications features like ad banners and geolocation.

You can read more about the project and view my paper in my LinkedIn profile in the education section.

Bailiff notifications pricelist

This android application is a useful tool for the judicial officers (court bailiffs) and also their customers and lawyers to find through their smartphones, using the search field feature, the prices for the service of legal documents in the region of Athens and the surroundings (Attiki).
